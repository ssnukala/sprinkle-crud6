ISSUES FIXED
===================================================================

1. ✅ Breadcrumbs disappeared from crud6/groups and crud6/groups/1
   - Root cause: Page title set asynchronously after schema load
   - Fix: Set page.title immediately, update after schema loads
   
2. ✅ Users table shows "0 - 0 of 0" on crud6/groups/1 detail page
   - Root cause: Schema not awaited, template renders too early
   - Fix: Use onMounted with async/await, add loading spinner
   
3. ✅ Edit Group button requires 2 clicks to launch modal
   - Root cause: uk-drop-close on placeholder closes dropdown
   - Fix: Remove uk-drop-close from placeholder button
