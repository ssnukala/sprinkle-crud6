Implementation Details

  • debug.ts - Debug utility with sessionStorage caching
  • ConfigAction.php - Backend endpoint at /api/crud6/config
  • plugins/crud6.ts - Auto-initializes debug mode on app load
  • All console.log/warn/error replaced with debugLog/debugWarn/debugError

How It Works:

  1. On first page load, initDebugMode() is called
  2. Checks sessionStorage for cached debug_mode value
  3. If not cached, fetches from /api/crud6/config (one API call per session)
  4. Stores value in sessionStorage for future use
  5. Falls back to environment variable VITE_DEBUG_MODE if API fails
